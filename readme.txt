== Description ==

!teoola affords a "new insigh" into an address book and creates a "word of mouth" recommendation tool on your Smartphone.

== Installation ==
=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'teoola'
3. Install !teoola once it appears
4. Activate !teoola from your Plugins page.
5. Go to "after activation" below.

=== Manually ===

1. Upload the `teoola` folder to the `/wp-content/plugins/` directory
2. Activate the !teoola plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

=== After activation ===

1. You should see the '!teoola' menu in the dashboard sidebar.
2. From there you can log in to your account.
3. You're done!
